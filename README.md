# Chat-Bot
__a simple chat bot__

#### How To Run
Open the Terminal and paste the commands below
```shell
git clone https://github.com/SystemWhoami/Chat-Bot.git
cd Chat-Bot
chmod +x chatter_bot.py
```
to run the bot, go to the file directory and type
```sh
./chatter_bot.py

```
